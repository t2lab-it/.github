# t2lab-it

東京理科大学 創域理工学部 機械航空宇宙工学科 [塚原研究室](https://www.rs.tus.ac.jp/~t2lab/index-j.html)の情報理論グループ（メンター：[荒木亮](https://ryo-araki.github.io/)）が使用するGitHub Organizationです．
